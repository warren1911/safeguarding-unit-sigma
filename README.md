# Safeguarding Unit (SU) and SIGMA

This repository provides canonical documentation for the **Safeguarding Unit (SU)**, a structured unit of measurement used to express safeguarding risk, capability, and protective effectiveness, and **SIGMA** — *Safeguarding Intelligence, Governance & Measurement Authority* — which defines the governance and analytical context in which safeguarding measurement is applied.

The materials are published in a neutral, technical format to support clarity, auditability, and responsible discussion. Publication of documentation does not constitute an open-standard dedication or transfer of intellectual property rights.
